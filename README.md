> gulp plugin for releasing npm module


## Install

```sh
$ npm install --save gulp-npm-release
```


## Usage

```js
var gulp-npm-release = require('gulp-npm-release');

gulp-npm-release('Rainbow');
```

```sh
$ npm install --global gulp-npm-release
$ gulp-npm-release --help
```
